Python基础学习大纲（详细版）

第一部分：Python入门

  • Python简介与特点

    Python由Guido van Rossum于1991年发布，主要特点：

    • 解释型语言 - 无需编译，直接运行
    • 动态类型 - 变量无需声明类型
    • 跨平台 - 支持Windows、Linux、Mac等
    • 丰富的标准库 - "电池包含"哲学
    • 多种编程范式 - 面向对象、函数式、过程式

    Python主要应用领域：

    • Web开发（Django, Flask）
    • 数据分析（Pandas, NumPy）
    • 人工智能（TensorFlow, PyTorch）
    • 自动化运维
    • 科学计算

    Python版本选择：

    Python 2.x (已停止维护)
Python 3.x (推荐使用3.8+版本)
  • Python安装与环境配置

    安装步骤：

    1. 从Python官网下载安装包（https://www.python.org/downloads/）
    2. 运行安装程序（Windows注意勾选"Add Python to PATH"）
    3. 验证安装： 
      # 命令行输入

# 或
python3 --version

    VS Code 轻量级编辑器+Python插件
    Jupyter Notebook 交互式笔记本，适合数据分析

  • Python基础语法规则

    Python语法特点：

    • 使用缩进表示代码块（通常4个空格）
    • 不需要分号结尾（但可以加）
    • 单行注释用#，多行注释用三引号 ''' 或 """
    • 大小写敏感

    代码示例：

    # 这是一个注释
'''
这是多行注释
可以写多行
'''

# 变量命名
my_variable = 10
MY_CONSTANT = 3.14  # 约定全大写表示常量

# 条件语句
if my_variable > 5:
    print("大于5")
else:
    print("小于等于5")

# 函数定义
def greet(name):
    """这是一个文档字符串"""
    return f"Hello, {name}!"

print(greet("Alice"))

    Python编码规范（PEP 8）：

    • 模块名：小写字母，可使用下划线（my_module.py）
    • 类名：大驼峰（MyClass）
    • 函数/变量名：小写字母，下划线分隔（my_function）
    • 每行不超过79个字符
    • 运算符两侧各留一个空格

2. 流程控制

  • 条件语句（if-elif-else）

    基本语法：

    # 简单if
age = 18
if age >= 18:
    print("成年人")

# if-else
if age >= 18:
    print("成年人")
else:
    print("未成年人")

# if-elif-else
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")

# 三元表达式
result = "通过" if score >= 60 else "不通过"

    条件表达式：

    • 比较运算符：==, !=, >, <, >=, <=
    • 逻辑运算符：and, or, not
    • 成员运算符：in, not in
    • 身份运算符：is, is not

    复杂条件示例：

    # 检查用户名和密码
username = "admin"
password = "123456"

if username == "admin" and password == "123456":
    print("登录成功")
else:
    print("用户名或密码错误")

# 检查数字范围
num = 15
if 10 <= num <= 20:
    print("数字在10到20之间")

# 检查列表包含
fruits = ['apple', 'banana', 'orange']
if 'apple' in fruits:
    print("有苹果")
  • 循环结构（for, while）

    for循环：

    # 遍历序列
fruits = ['apple', 'banana', 'orange']
for fruit in fruits:
    print(fruit)

# 遍历字符串
for char in "Python":
    print(char)

# 使用range
for i in range(5):      # 0到4
    print(i)

for i in range(1, 6):   # 1到5
    print(i)

for i in range(0, 10, 2): # 0,2,4,6,8
    print(i)

    while循环：

    # 基本while
count = 0
while count < 5:
    print(count)
    count += 1

# 无限循环+break
while True:
    user_input = input("输入quit退出: ")
    if user_input == "quit":
        break
    print(f"你输入了: {user_input}")

# continue示例
num = 0
while num < 10:
    num += 1
    if num % 2 == 0:
        continue  # 跳过偶数
    print(num)

    循环中的else：

    # for-else: 循环正常结束时执行
for i in range(5):
    print(i)
else:
    print("循环完成")

# while-else
count = 0
while count < 3:
    print(count)
    count += 1
else:
    print("循环结束")

# 如果循环被break中断，else不会执行
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("这不会执行")
  • 循环控制（break, continue, pass）

    break语句：

    # 查找第一个偶数
numbers = [1, 3, 5, 8, 10, 12]
for num in numbers:
    if num % 2 == 0:
        print(f"找到第一个偶数: {num}")
        break

# while循环中的break
while True:
    answer = input("输入quit退出: ")
    if answer == "quit":
        break
    print(f"你输入了: {answer}")

    continue语句：

    # 打印1-10的奇数
for i in range(1, 11):
    if i % 2 == 0:
        continue
    print(i)

# 跳过特定元素
fruits = ['apple', 'banana', 'orange', 'grape']
for fruit in fruits:
    if fruit == 'orange':
        continue
    print(fruit)

    pass语句：

    # 空函数/类定义
def empty_function():
    pass  # 什么都不做，但语法需要

class EmptyClass:
    pass

# 占位符
for i in range(5):
    if i == 3:
        pass  # 以后实现
    else:
        print(i)

    循环控制对比：

    语句 作用 示例
    break 立即退出整个循环 找到目标后终止搜索
    continue 跳过当前迭代，继续下一次 跳过某些特定元素
    pass 空操作，语法占位符 暂时不实现某代码块

第三部分：Python数据结构

1. 列表(List)

  • 列表操作与方法

    列表创建：

    # 空列表
empty_list = []
empty_list = list()

# 初始化列表
numbers = [1, 2, 3, 4, 5]
fruits = ['apple', 'banana', 'orange']
mixed = [1, 'hello', 3.14, True]

    列表操作：

    # 索引和切片
lst = [10, 20, 30, 40, 50]
lst[0]    # 10
lst[-1]   # 50
lst[1:3]  # [20, 30]
lst[::2]  # [10, 30, 50]

# 修改元素
lst[1] = 99  # [10, 99, 30, 40, 50]

# 列表连接
lst1 = [1, 2, 3]
lst2 = [4, 5, 6]
combined = lst1 + lst2  # [1, 2, 3, 4, 5, 6]

# 列表重复
repeated = lst1 * 2  # [1, 2, 3, 1, 2, 3]

    列表方法：

    lst = [1, 2, 3]

# 添加元素
lst.append(4)        # [1, 2, 3, 4]
lst.insert(1, 1.5)   # [1, 1.5, 2, 3, 4]
lst.extend([5, 6])   # [1, 1.5, 2, 3, 4, 5, 6]

# 删除元素
lst.remove(1.5)      # 移除第一个匹配项
popped = lst.pop()   # 移除并返回最后一个元素(6)
popped = lst.pop(1)  # 移除并返回索引1的元素(2)
del lst[0]           # 删除索引0的元素

# 其他操作
lst.index(3)         # 返回第一个3的索引
lst.count(3)         # 统计3出现的次数
lst.sort()           # 原地排序
lst.reverse()        # 原地反转
lst.copy()           # 浅拷贝
lst.clear()          # 清空列表

    列表推导式：

    # 创建平方数列表
squares = [x**2 for x in range(10)]

# 带条件的推导式
even_squares = [x**2 for x in range(10) if x % 2 == 0]

# 嵌套推导式
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flattened = [num for row in matrix for num in row]  # [1,2,3,4,5,6,7,8,9]

2. 元组(Tuple)

  • 元组操作与特性

    元组特性：

    • 不可变序列
    • 通常用于存储异构数据
    • 可以作为字典的键
    • 比列表更节省内存

    元组创建：

    # 空元组
empty_tuple = ()
empty_tuple = tuple()

# 单元素元组(注意逗号)
single = (1,)  # 必须加逗号
not_single = (1)  # 这不是元组，是整数1

# 多元素元组
colors = ('red', 'green', 'blue')
numbers = (1, 2, 3, 4, 5)

# 不加括号也可以(但不推荐)
mixed = 'a', 1, True

    元组操作：

    t = (10, 20, 30, 40, 50)

# 索引和切片
t[0]    # 10
t[-1]   # 50
t[1:3]  # (20, 30)

# 元组解包
a, b, c = (1, 2, 3)  # a=1, b=2, c=3

# 交换变量
x, y = 10, 20
x, y = y, x  # 使用元组解包交换

# 作为返回值
def get_stats(numbers):
    return min(numbers), max(numbers), len(numbers)

min_val, max_val, count = get_stats([1, 2, 3, 4, 5])

    元组方法：

    t = (1, 2, 3, 2, 4, 2)

t.count(2)  # 3 (统计2出现的次数)
t.index(3)  # 2 (返回第一个3的索引)

    元组与列表比较：

    特性 列表(List) 元组(Tuple)
    可变性 可变 不可变
    语法 方括号[] 圆括号()
    性能 较慢 较快
    用途 存储同构数据 存储异构数据
    作为字典键 不可以 可以

3. 字典(Dictionary)

  • 字典操作与方法

    字典特性：

    • 键值对集合
    • 键必须是不可变类型（字符串、数字、元组）
    • 值可以是任意类型
    • 无序（Python 3.7+保持插入顺序）

    字典创建：

    # 空字典
empty_dict = {}
empty_dict = dict()

# 初始化字典
person = {'name': 'Alice', 'age': 25, 'city': 'New York'}
grades = dict(math=90, english=85, history=92)

# 字典推导式
squares = {x: x**2 for x in range(1, 6)}  # {1:1, 2:4, 3:9, 4:16, 5:25}

    字典操作：

    d = {'name': 'Alice', 'age': 25}

# 访问元素
d['name']  # 'Alice'
d.get('age')  # 25
d.get('salary', 0)  # 键不存在返回默认值0

# 添加/修改元素
d['city'] = 'New York'  # 添加
d['age'] = 26           # 修改

# 删除元素
del d['city']  # 删除键'city'
age = d.pop('age')  # 删除并返回值
d.clear()  # 清空字典

    字典方法：

    d = {'a': 1, 'b': 2, 'c': 3}

# 获取所有键
d.keys()    # dict_keys(['a', 'b', 'c'])

# 获取所有值
d.values()  # dict_values([1, 2, 3])

# 获取键值对
d.items()   # dict_items([('a', 1), ('b', 2), ('c', 3)])

# 更新字典
d.update({'b': 20, 'd': 4})  # {'a':1, 'b':20, 'c':3, 'd':4}

# 设置默认值
d.setdefault('e', 5)  # 如果'e'不存在，设置为5

    字典遍历：

    # 遍历键
for key in d:
    print(key)

for key in d.keys():
    print(key)

# 遍历值
for value in d.values():
    print(value)

# 遍历键值对
for key, value in d.items():
    print(f"{key}: {value}")

4. 集合(Set)

  • 集合操作与方法

    集合特性：

    • 无序的唯一元素集合
    • 可变（frozenset是不可变版本）
    • 支持数学集合运算
    • 元素必须是不可变类型

    集合创建：

    # 空集合(必须用set())
empty_set = set()

# 初始化集合
numbers = {1, 2, 3, 4, 5}
letters = set('hello')  # {'h', 'e', 'l', 'o'}

# 集合推导式
squares = {x**2 for x in range(1, 6)}  # {1, 4, 9, 16, 25}

    集合操作：

    a = {1, 2, 3}
b = {3, 4, 5}

# 并集
a | b  # {1, 2, 3, 4, 5}
a.union(b)

# 交集
a & b  # {3}
a.intersection(b)

# 差集
a - b  # {1, 2}
a.difference(b)

# 对称差集(只在其中一个集合中的元素)
a ^ b  # {1, 2, 4, 5}
a.symmetric_difference(b)

    集合方法：

    s = {1, 2, 3}

# 添加元素
s.add(4)       # {1, 2, 3, 4}
s.update([5, 6]) # {1, 2, 3, 4, 5, 6}

# 删除元素
s.remove(6)    # 如果6不存在会报错
s.discard(5)   # 如果5不存在不会报错
popped = s.pop()  # 移除并返回任意元素
s.clear()      # 清空集合
