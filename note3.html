Java基础学习大纲（详细版）

第一部分：Java入门

  • Java简介与发展历史

    Java由Sun Microsystems公司于1995年推出，后被Oracle收购。主要版本发展：

    • Java 1.0 (1996)
    • Java 5 (2004) - 重大更新，加入泛型等
    • Java 8 (2014) - 加入Lambda表达式
    • Java 11 (2018) - LTS版本
    • Java 17 (2021) - 最新LTS版本

    Java特点：跨平台、面向对象、自动内存管理、多线程支持等。

  • Java平台与JVM原理

    Java平台组成：

    • Java虚拟机(JVM) - 执行字节码
    • Java运行时环境(JRE) - 包含JVM和核心类库
    • Java开发工具包(JDK) - 包含JRE和开发工具

    JVM工作原理：

    Java源代码 → 编译器 → 字节码(.class) → JVM → 机器码

    JVM内存区域：方法区、堆、虚拟机栈、本地方法栈、程序计数器

  • JDK安装与环境配置

    安装步骤：

    1. 从Oracle官网下载JDK安装包
    2. 运行安装程序
    3. 配置环境变量： 
      # Windows系统
1. 右键"此电脑" → 属性 → 高级系统设置 → 环境变量
2. 在系统变量中新建JAVA_HOME，值为JDK安装路径（如C:\Program Files\Java\jdk-17）
3. 编辑Path变量，添加%JAVA_HOME%\bin

# Linux/Mac系统
在~/.bashrc或~/.zshrc中添加：
export JAVA_HOME=/path/to/jdk
export PATH=$JAVA_HOME/bin:$PATH
    4. 验证安装：
      java -version

2. 流程控制

  • 条件语句（if-else, switch）

    if-else语句：

    // 简单if
if (score >= 60) {
    System.out.println("及格");
}

// if-else
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}

    switch语句（Java 12+支持箭头语法）：

    int day = 3;
String dayName;

// 传统语法
switch (day) {
    case 1: dayName = "星期一"; break;
    case 2: dayName = "星期二"; break;
    case 3: dayName = "星期三"; break;
    default: dayName = "未知";
}

// Java 12+新语法
dayName = switch (day) {
    case 1 -> "星期一";
    case 2 -> "星期二";
    case 3 -> "星期三";
    default -> "未知";
};

    switch表达式（Java 14+）：

    String season = switch (month) {
    case 12, 1, 2 -> "冬季";
    case 3, 4, 5 -> "春季";
    case 6, 7, 8 -> "夏季";
    case 9, 10, 11 -> "秋季";
    default -> "无效月份";
};
  • 循环结构（for, while, do-while）

    for循环：

    while循环：

    do-while循环：

    循环控制：

    // break示例 - 跳出循环
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
    System.out.println(i);
}

// continue示例 - 跳过当前迭代
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) continue;
    System.out.println(i);  // 只打印奇数
}
  • 跳转语句（break, continue, return）

    break语句：

    // 1. 跳出循环
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // 当i等于5时跳出循环
    }
    System.out.println(i);
}

// 2. 带标签的break（跳出多层循环）
outerLoop:
for (int i = 0; i < 5; i++) {
    for (int j = 0; j < 5; j++) {
        if (i * j > 6) {
            break outerLoop;  // 直接跳出外层循环
        }
        System.out.println(i + " * " + j + " = " + (i * j));
    }
}

    continue语句：

    // 1. 跳过当前迭代
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;  // 跳过偶数
    }
    System.out.println(i);  // 只打印奇数
}

// 2. 带标签的continue
outerLoop:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == j) {
            continue outerLoop;  // 跳过外层循环当前迭代
        }
        System.out.println(i + "," + j);
    }
}

    return语句：

    // 1. 返回值
public int add(int a, int b) {
    return a + b;  // 返回两数之和
}

// 2. 提前结束方法
public void printPositive(int num) {
    if (num <= 0) {
        return;  // 如果数字非正，提前结束方法
    }
    System.out.println("正数: " + num);
}

    简单TCP客户端/服务器示例：

    // 服务器端
try (ServerSocket server = new ServerSocket(8080);
     Socket client = server.accept();
     PrintWriter out = new PrintWriter(client.getOutputStream(), true);
     BufferedReader in = new BufferedReader(new InputStreamReader(client.getInputStream()))) {
    
    String inputLine;
    while ((inputLine = in.readLine()) != null) {
        System.out.println("收到: " + inputLine);
        out.println("服务器回复: " + inputLine);
    }
}

// 客户端
try (Socket socket = new Socket("localhost", 8080);
     PrintWriter out = new PrintWriter(socket.getOutputStream(), true);
     BufferedReader in = new BufferedReader(new InputStreamReader(socket.getInputStream()));
     BufferedReader stdIn = new BufferedReader(new InputStreamReader(System.in))) {
    
    String userInput;
    while ((userInput = stdIn.readLine()) != null) {
        out.println(userInput);
        System.out.println("服务器说: " + in.readLine());
    }
}
  • Java数据库连接（JDBC）

    JDBC基本步骤：

    // 1. 加载驱动（JDBC 4.0+自动加载）
Class.forName("com.mysql.cj.jdbc.Driver");

// 2. 建立连接
String url = "jdbc:mysql://localhost:3306/mydb?useSSL=false";
String user = "root";
String password = "123456";
try (Connection conn = DriverManager.getConnection(url, user, password)) {
    
    // 3. 创建Statement
    Statement stmt = conn.createStatement();
    
    // 4. 执行查询
    ResultSet rs = stmt.executeQuery("SELECT * FROM users");
    
    // 5. 处理结果
    while (rs.next()) {
        int id = rs.getInt("id");
        String name = rs.getString("name");
        System.out.println(id + ": " + name);
    }
    
    // 6. 关闭资源（try-with-resources自动关闭）
}

  • Java GUI编程（Swing/JavaFX）

    import javax.swing.*;

public class SwingDemo {
    public static void main(String[] args) {
        // 创建主窗口
